## Enablement Toolkit — AI & Systems Workshops

**Curator:** Zohaib Cheema  
**Audience:** University tech clubs, bootcamps, internal guilds  
**Bundle Contents:** Slide decks, starter repos, facilitation notes, follow-up templates

---

### 1. Workshop Catalog
| Title | Duration | Outcome |
|-------|----------|---------|
| GenAI Builder Lab | 90 min | Participants ship a planner/builder workflow with eval guardrails |
| Cloud Ops Simulator | 75 min | Teams run a blue/green rollout, read dashboards, publish RCA |
| Inclusive Design Sprint | 60 min | Designers run heuristic review + implement motion/contrast fixes |

Each workshop includes:
- Slide deck (Google Slides + PDF)
- Live demo repo (Node/React)
- Facilitation script with time boxes
- Retro + feedback form

---

### 2. Facilitation Notes (Snippet)
```
00:00 - 00:05 Icebreaker (“Tell us the last bug you fixed”)

00:45 - 01:00 Retro + next steps
```
